# GachaCalc (静的版)

ブラウザだけで動作するガチャ確率計算ツールです。Flask サーバーを使わない完全静的サイトなので、GitHub Pages や Cloudflare Pages にそのまま置くだけで公開できます。

## 特徴

- **少なくとも1つ以上当たる確率**、**m個以上当たる確率**を即時計算。
- **指定した目標確率を満たすのに必要な試行回数**を表示。
- スライダーとボタンで排出率・試行回数・目標確率を直感的に調整。
- 0〜5 回当たる確率の分布を同時に確認可能。

## 使い方

1. `index.html` をブラウザで開きます。ローカルファイルとして開くだけで動きます。
2. 排出率、試行回数、欲しい個数、達成したい確率を入力すると結果がリアルタイムに更新されます。

## デプロイ（GitHub Pages 例）

1. 本リポジトリを GitHub に push します。
2. リポジトリ設定の「Pages」で **Source: Deploy from a branch** を選び、ブランチを `main`、フォルダーを `/root` に設定します。
3. 数十秒待つと `https://<ユーザー名>.github.io/<リポジトリ名>/` で公開されます。

同様に Cloudflare Pages でも、リポジトリを連携してビルドステップなしでデプロイできます。サーバー代ゼロ・クレカ登録不要で公開可能です。

## ファイル構成

```
GACHACALC/
├── index.html          # メインページ（静的）
└── static/
    ├── main.js         # 計算ロジックとイベントハンドラ（ブラウザ上で実行）
    └── style.css       # デザイン
```
